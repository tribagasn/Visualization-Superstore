# Visualization-Superstore
Profitability & Discount Impact Analysis Dashboard is a data analytics project built to uncover how discount strategies influence business profitability. Using the Sample Superstore dataset, this dashboard integrates various visualizations into one interactive report, enabling users to explore sales performance, profit distribution, product performance, regional contributions, and the relationship between discounts and profit. The insights generated from this project can help businesses optimize discount policies, improve profitability, and make more informed strategic decisions through data-driven analysis.
